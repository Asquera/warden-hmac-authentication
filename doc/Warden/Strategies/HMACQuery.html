Overview

Implements query-based hmac authentication for warden. The strategy is registered as `:hmac_query` in the warden strategy list.

def valid?
  valid = auth_info.include? "signature"
  valid = valid && has_timestamp? if check_ttl?
  valid = valid && has_nonce? if nonce_required?
  valid
end
